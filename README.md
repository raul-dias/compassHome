# compassHome
An arduino based compass that will always point to 'home'

## Components List
1. Arduino NANO
2. Gps Module
3. Compass Module
4. Stepper Motor
5. Batter with 5v controller
       
## Connections
### GPS
1. vcc - 5V
2. GND - GND
3. rx - D3
4. tx - D4

### Compass
1. vcc - 5v
2. gnd - GND
3. SCL - A5
4. SDA - A4

### Stepper
1. vcc - external 5v
2. gnd - GND
3. In1 - 8
4. In2 - 9
5. In3 - 10
6. In4 - 11
