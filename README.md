# compassHome
An arduino based compass that will always point to 'home'

## Components List
1. Arduino NANO
2. Gps Module
3. Compass Module
4. Stepper Motor
5. Batter with 5v controller
       
## Connections  
### GPS              
vcc - 5V        
GND - GND      
RX - D3         
tx - D4         

### Compass
vcc - 5v
gnd - GND
SCL - A5
SDA - A4

### Stepper
vcc - external 5v
gnd - GND
In1 - 8
In2 - 9
In3 - 10
In4 - 11
